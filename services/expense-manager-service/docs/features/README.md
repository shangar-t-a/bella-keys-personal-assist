# Expense Manager Service Features

This documentation covers the core features of the Expense Manager Service (EMS).

## Features

### 1. Savings Buckets & Allocations
Manage target sub-savings goals within accounts. Supports safe auto-refunds on deletion, root account protection, and an audited transaction ledger.
* [Guide](savings-bucket.md)

### 2. Accounting Periods
Track and isolate financial records within discrete monthly periods.
* [Guide](accounting-periods.md)

### 3. Spending Entries & Ledger
Transaction logging for daily expenses and income. Supports server-side sorting, filtering, and pagination to optimize frontend performance.
* [Guide](spending-ledger.md)

### 4. Monthly Planner & Dashboard
Displays summaries, target allocations, and category-wise spending charts.
* [Guide](monthly-planner.md)
